project status
--------------

- spontaneous idea
- had some fun coding
- works so far

Not all ChoiceScript parameters are implemented yet.  I don't know if
they ever will be.

Note that text nodes are shortened to *T[character_count]*.  The first
part of the text will be displayed as a tooltip.
